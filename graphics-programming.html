Graphics Programming

A C++ and OpenGL project focused on rendering systems, shaders, lighting, and real-time graphics programming.

Project Overview

This project was developed as part of my Graphics Programming coursework using C++ and OpenGL. The focus of the project was to explore real-time rendering systems and gain practical experience with lower-level graphics programming concepts.

Through this project, I implemented rendering features such as model loading, texture mapping, lighting, and shader-based visual effects. The work helped develop a stronger understanding of the graphics pipeline and how rendering systems contribute to game development.

Project Media

Demonstration of rendering features including models, textures, shaders, and lighting systems.

Key Features

Model Rendering

Loaded and rendered 3D models within an OpenGL scene.

Texture Mapping

Applied textures to objects to improve scene detail and visual quality.

Shaders

Used shader-based rendering to control visual effects and graphical output.

Lighting

Implemented lighting systems to improve realism and scene readability.

Rendering Pipeline

Worked directly with OpenGL pipeline concepts and real-time rendering structure.

Technical Debugging

Resolved low-level graphics issues related to shaders, models, and rendering behaviour.

Tools & Technologies

Language: C++
Graphics API: OpenGL
Tools: Visual Studio, shaders, model loading, texture handling

What I Learned

This project improved my understanding of real-time graphics programming and the technical processes behind rendering systems in games.

It also strengthened my debugging skills by requiring me to identify and resolve problems in lower-level systems such as shaders, rendering setup, and asset handling.
