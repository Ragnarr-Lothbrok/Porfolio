Timeline Disaster

A level-based game set across different points in history, developed as part of my HND project.

Project Overview

Timeline Disaster is a level-based game developed during my HND, focused on gameplay progression across multiple historical settings. Each level represents a different point in history, with enemies and environments themed around that time period.

The project includes three distinct levels, each designed with its own setting, enemy types, and gameplay challenges. The aim was to create a structured experience that evolves as the player progresses through different timelines.

Project Media

Demonstration of gameplay across different historical levels and enemy encounters.

Key Features

Multiple Levels

Three levels set in different historical time periods.

Themed Enemies

Enemies designed to match each timeline and setting.

Progression

Structured level progression with increasing challenge.

Game Design

Focus on level structure, pacing, and player experience.

Environment Design

Different environments representing historical settings.

Core Systems

Basic gameplay systems supporting movement, interaction, and progression.

Tools & Technologies

Engine: Unity
Language: C#
Focus: Game design, level structure, progression systems

What I Learned

This project helped develop my understanding of level design, progression, and how to structure a complete game experience across multiple stages.

It also improved my ability to design gameplay around themes and environments, ensuring each level felt distinct while still contributing to the overall experience.
